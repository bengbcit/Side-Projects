# LINE Mini App Project

This folder is dedicated to developing **LINE Mini Apps** using the **LIFF (LINE Front-end Framework)**.

### 🎯 Goals
- Integrate with LINE's social features.
- Develop specialized tools for users in Japan.
- Experiment with AI-driven development for the LINE ecosystem.

### 🛠️ Tech Stack
- HTML5 / CSS3 / JavaScript
- LIFF SDK
- AI Tools: Claude Code
